DESCRIPTION:
This project is a webhook project used for receiving notifications of new transactions created by IMTOs. 
The transactions received are stored in the WorldRemitForCashpickup database in the TransactionsRequests table.

LANGAUAGE:
C# ASP.NET

PROJECT TYPE:
WEB API Project
